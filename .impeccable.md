# Vex&Co Lab — Design Context

## Identity
Boutique strategic PM Lab. "Quiet Luxury" aesthetic.
Not a SaaS dashboard. Think: high-end consulting firm's
private portal meets editorial magazine.

## Typography
- Headings: Cormorant Garamond (serif, editorial weight)
- Body: Inter (clean, modern, light weight)
- Never: Arial, Helvetica, system defaults

## Color Palette
- Background: Off-white #F9F8F6 (ql-offwhite)
- Text: Charcoal #1A1A1A (ql-charcoal)
- Accent: Warm bronze #8B7355 (ql-accent) — sparingly
- Surfaces: Cream #F3F1ED, Sand #E8E4DD
- Muted: #B8B2A8 for secondary text
- NO bright colors, NO gradients, NO shadows

## CSS Class System (ql-* prefix)
- `ql-h1`, `ql-h2`, `ql-h3` — heading hierarchy (Cormorant Garamond)
- `ql-body` — body text (Inter, text-ql-slate)
- `ql-label` — uppercase micro-labels (tracking-widest, text-ql-muted)
- `ql-caption` — small uppercase captions
- `ql-card` — flat surface card (no shadow, subtle border)
- `ql-card-flat` — borderless flat card
- `ql-btn-primary` — charcoal fill button
- `ql-btn-secondary` — ghost outlined button
- `ql-btn-ghost` — text-only button
- `ql-input` — ghost input (transparent, bottom border only)
- `ql-status-thinking` — pulsing dot (agent working)
- `ql-status-active` — green dot (agent online)
- `ql-page` — page wrapper with standard padding

## Principles
- Zero-UI: interface disappears, content is protagonist
- Separation by typography hierarchy and spacing, not borders
- No modals that block — everything non-blocking
- Monocromatic avatar system (cream/charcoal/accent)
- MoSCoW badges: subtle, not loud
- Loading: pulsing dot + italic text, never spinners

## Anti-patterns
- NO colored avatar circles (green, blue, purple)
- NO gray Tailwind borders (border-gray-200)
- NO card-in-card nesting
- NO aggressive spinners (Loader2)
- NO modals that gray out the background
- NO Inter as heading font (only body)
- NO `shadow-md` or any drop shadows
- NO colored backgrounds on inputs
